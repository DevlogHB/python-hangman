## Hangman Game

파이썬 문법으로 Hangman Game project 만들기

#### 설명

- 무작위로 단어가 설정되며, 사용자가 입력한 알파벳이 단어에 있는지 확인합니다.
- 단어에 알파벳이 포함되어 있지 않으면 기회는 줄어듭니다. (기회:6번)
- 단어를 맞추면 "You Win" 메시지가 출력되며 종료됩니다.
